Why Lume Stats Isn't Coming

Lume Stats started as a passion project — something I wanted to build for myself and share with others. A sleek, beautiful, modern performance monitor for PC users.

But building a full system-level monitoring app alone is extremely challenging. It requires deep Windows API work, performance tracking hooks, kernel permissions, and constant debugging on different hardware setups.

The biggest issue: Antivirus false flags.
Even when nothing malicious is inside the program, Windows Defender and other AV engines commonly flag tools that access GPU, CPU and system metrics. Fixing these false positives requires certification, code signing, and a ton of money I simply don’t have yet.

And since I’m working completely alone — no team, no testers, no funding — developing it would take an incredible amount of time and stress.

So for now, Lume Stats is on an indefinite freeze. Maybe one day it will come back — but not anytime soon...
